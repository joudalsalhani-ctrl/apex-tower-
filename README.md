# Apex One Tower Web App

A mobile-first interactive 3D architectural concept.

## Features
- Touch drag rotation
- Pinch zoom
- Tap any floor
- Floor selector
- Hide above
- Isolate floor
- Exploded view
- Day/night mode
- Concept interior floor plan

## Deploy on Vercel
1. Extract this ZIP.
2. Go to Vercel and create a new project.
3. Upload the extracted folder.
4. Deploy as a static site.

## Deploy on GitHub Pages
1. Create a new repository.
2. Upload all files from this folder.
3. Open Settings > Pages.
4. Select the main branch and root folder.
5. Save and open the generated Pages URL.

Internet access is required because Three.js loads from a CDN.
